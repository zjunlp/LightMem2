# Saved Experiment Outputs

This tree is reserved for local PinchBench run artifacts produced from the
migrated mainline harness.

It keeps the directory structure that the wrappers expect, but committed run
outputs should stay out of version control unless a specific artifact is needed
for a debugging record.

Current layout:

- `continuous/method/raw/`
- `isolated/method/raw/`
- `logs/`
- `reports/`
